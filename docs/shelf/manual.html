What's this?

通販会社「千趣会」のカバンのネット通販用のWebアプリケーションです。

Selling Point

「機能的で使いやすい」をコンセプトに開発をしました。

  • 誰にでも優しい直感的な操作。
  • EaseOutを用いた滑らかな回転。
  • 商品の検索をサポートする実用的なソート機能。
  • PCでもNexus7でも快適に動作。
    • WebGLの動作するWebブラウザさえあれば動作可能。
    • タブレットのフリック操作に対応。

Usage

棚の回転方法

棚を回転させる方法は3種類あります。

  • 棚の左右の矢印アイコンをクリックする。
  • 画面下のバーの数字をクリックする。
  • 棚をドラッグ(タブレットならフリック)する。

商品の詳細情報

カバンのパネルをクリックすると、パネルが手前にポップアップし、商品名や値段などの詳細情報を見ることができます。

ソート機能

画面上部のナビゲーションバーにあるアイコンの右隣に、「id」「price」「culor」の3項目があります。

これらの項目をクリックすることで、それぞれ「商品ID」「値段」「色」によってカバンをソートできます。

もちろん、昇順ソートと降順ソートの両方に対応しています。

Configuration

画面上部のナビゲーションバーにあるアイコンをクリックすると、設定画面が開きます。

ここから、次の項目を設定出来ます。

  • アニメーションの動きにEaseOutを効かせるかどうか。
  • ステータスバーの種類。

Development Information

開発には次のようなものを使いました。

  • MatrixEngine (WebGL)
  • JavaScript (jQuery)
  • HTML5, CSS3
  • Bootstrap
  • C++, OpenCV (色相の計算に使用)
