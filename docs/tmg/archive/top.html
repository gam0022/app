About

What's this?

Twitter ID から簡単に名刺を作成できる Web サービスです。

Features
  • わずか2ステップで簡単作成！
    1. Twitter IDを入力。
    2. デザインを選択。
  • ユーザ認証しなくても使用可能。
  • Canvas を用いた描画により、デザイン選択した瞬間に名刺が作れる！
  • マルコフ連鎖で直近200件のツイートをカオスに要約。
  • 完成した名刺は PNG 形式で保存、Twitterで共有できる！
